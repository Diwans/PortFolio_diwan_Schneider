Projet de création d'une page web dynamique pour une photographe. (projet formation)


Ma mission dans ce projet était de rendre cette page web dynamique avec le langage JavaScript et d'y intégrer l'utilisation d'une API pour recupérer des données depuis celle ci au lieu de les intégrer directement via le HTML. J'ai aussi creer la page de connexion pour l'administrateur ( client ) du site a partir de zero et j'ai également developpé une modale permettant de gerer depuis le site les medias que le client voudrais soit ajouter, consulter, supprimer ou modifier.

Difficultés rencontrées durants le projet :

  • apprentissage du langage JavaScript

Compétences developpées :

  • manipulations des éléments du DOM avec JS
  • utilisation d'une API avec JS
  • rendre une page web dynamique
