Projet de création d'une page web dynamique pour une photographe. (projet formation)


Ma mission dans ce projet était de rendre cette page web dynamique avec le langage Javascript et d'y intégrer l'utilisation d'une API pour récupérer des données depuis celle-ci au lieu de les intégrer directement via le HTML. J'ai aussi créer la page de connexion pour l'administrateur ( client) du site à partir de zéro et j'ai également développé une modale permettant de gérer depuis le site les medias que le client voudrait soit ajouter, consulter, supprimer ou modifier.
